# Darwin Security Report Generator

A Python cybersecurity project that generates and saves security incident reports.

## Features

- Creates incident reports
- Assigns severity levels
- Saves reports to a text file
- Introduces incident documentation concepts

## Example

Input:

```text
Incident: Failed login attempts
Severity: High
```

Output:

```text
=== Security Report ===

Incident:
Failed login attempts

Severity:
High

Status:
Open

Analyst:
Darwin Brown
```
